Reach 1.0 — Campus ride booking web app for SafeRide. Built with React + Vite. Student and driver panels with real-time booking, QR verification, and Paystack wallet integration. PWA enabled.

# Reach 1.0 — Client

> Book your SafeRide. No cash. No stress.

Reach is a campus ride-booking web app built for SafeRide tricycles.
Students book rides from any campus park, pay digitally, and verify 
with a QR scan. Drivers manage bookings and withdrawals from a 
dedicated panel.

## Features

- Student and driver panels from a single login flow
- Park-based ride booking with fixed pricing
- Day and night ride pricing
- QR code ride verification
- Paystack wallet top-up and escrow system
- Real-time booking and driver status updates
- PWA — installable on any mobile device

## Tech Stack

- React + Vite
- Tailwind CSS
- Socket.io Client
- Paystack
- Progressive Web App (vite-plugin-pwa)
